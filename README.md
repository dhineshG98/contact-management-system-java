# CONTACT MANAGEMENT SYSTEM

## Project Overview

The Contact Management System is a console-based Java application that allows users to store and manage contact information efficiently. Users can add new contacts, view all contacts, search for contacts by name, and delete contacts through a simple menu-driven interface.

This project is designed to demonstrate the implementation of Object-Oriented Programming (OOP) concepts and Java Collections Framework.

---

## Features

- Add new contacts
- View all saved contacts
- Search contacts by name
- Delete contacts
- User-friendly menu-driven interface
- Efficient contact management using ArrayList

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## Contact Information Stored

Each contact contains:

- Name
- Phone Number
- Email Address
- Address

---

## Project Structure

### Contact Class
Stores contact details and provides methods to access and modify contact information.

### ContactManager Class
Manages all contact operations including:
- Add Contact
- View Contacts
- Search Contact
- Delete Contact

### ContactApp Class
Contains the main method and menu-driven user interface.

---

## How to Run

1. Compile the program:
   ```bash
   javac ContactApp.java
   ```

2. Run the program:
   ```bash
   java ContactApp
   ```

3. Select an option from the menu and manage contacts.
